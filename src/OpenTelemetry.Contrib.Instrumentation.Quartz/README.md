# QuartzNET Instrumentation for OpenTelemetry .NET

Automatically instruments the Quartz jobs from
[Quartz](https://www.nuget.org/packages/Quartz/).

## Installation

```shell
dotnet add package OpenTelemetry.Contrib.Instrumentation.Quartz
```
